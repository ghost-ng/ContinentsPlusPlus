üìã How to Use

1. Install Dependencies

cd map-preview-tool
npm install

2. Generate Map Previews

# Generate all map sizes
node src/index.js --all

# Generate a specific size
node src/index.js --size HUGE --seed 12345

# Generate with overlays
node src/index.js --size STANDARD --voronoi
node src/index.js --size LARGE --grid

3. View the Results

Open this HTML file in your browser to see the generated maps, or check the output/ folder for PNG files.

Note: This is a simplified preview tool. The actual in-game maps may differ slightly due to additional terrain processing, biome assignment, and feature placement that occurs in the full Civ VII map generation pipeline.

Available Options

  • --size: TINY, SMALL, STANDARD, LARGE, HUGE
  • --seed: Random seed (number) for reproducible maps
  • --output: Output file path
  • --grid: Show hexagonal grid overlay
  • --voronoi: Show Voronoi plate boundaries
  • --all: Generate all map sizes

Map Statistics

The generator targets ~68% water coverage (Earth-like) with 2-4 major continents and organic coastlines created through erosion simulation and coastal island generation.
